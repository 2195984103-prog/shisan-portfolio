## 六、动效规范

| 项目 | 填写 |
| --- | --- |
| 动效整体气质 | 克制 / 活泼 / 科技 / 品牌感 |
| 是否需要开场动画 | 是 / 否 |
| 是否需要滚动出现 | 是 / 否 |
| 是否需要轮播 | 是 / 否 |
| hover 效果 | 图片轻微放大 / 文字变色 / 阴影 / 无 |
| 禁止动效 | 弹跳 / 快速闪烁 / 复杂 3D / 过度发光 |
| 无障碍要求 | 支持 prefers-reduced-motion |

当前作品集示例：

```txt
动效要慢、轻、克制。可以使用渐入、轻微位移、轻微缩放，但不要弹跳、翻转、强光效。必须支持 prefers-reduced-motion。
```
